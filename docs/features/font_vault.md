# 기획 정의서: PPT Reference Vault

PPT 제작을 위한 참고 자료(폰트, 색상 팔레트, 레퍼런스 사이트)를 체계적으로 관리하고 활용할 수 있는 웹 서비스의 핵심 사양을 정의합니다.

## 1. 폰트 관리 기능 (Font Management)

### 요구사항 및 정책
- **눈누 사이트 연동**: 사용자가 상업적 무료 한글 폰트를 쉽게 내려받을 수 있도록 서비스 최상단에 **눈누(https://noonnu.cc/)** 사이트 바로가기 버튼을 고정 배치합니다.
- **다중 폰트 파일 업로드**:
  - 사용자는 폰트 등록 화면에서 여러 개의 폰트 파일(`.ttf`, `.otf`, `.woff`, `.woff2`)을 동시에 선택하여 일괄 업로드할 수 있습니다.
  - 업로드된 각 파일은 개별 폰트로 저장 및 관리됩니다.
- **서버 파일 관리**: 업로드된 모든 폰트 파일은 프로젝트 루트의 `/FONT` 디렉토리 아래에서 고유한 파일명으로 관리됩니다.
- **실시간 웹 폰트 미리보기**:
  - 등록된 폰트는 프론트엔드단에서 동적으로 CSS `@font-face`로 로드되어 화면에 렌더링됩니다.
  - 사용자가 입력창에 작성한 텍스트 문구가 등록된 모든 폰트의 서체로 실시간 적용되어 비교 테스트가 가능해야 합니다.
- **개발 코드 복사 및 삭제**:
  - 웹/PPT 기획 단계 등에서 빠르게 폰트 명칭을 사용할 수 있도록 해당 폰트 카드의 `font-family` CSS 복사 기능 및 폰트 삭제 기능을 제공합니다.

---

## 2. 색상 팔레트 관리 기능 (Color Palette)

### 요구사항 및 정책
- PPT 문서의 브랜드 아이덴티티 및 슬라이드 테마용 색상 조합을 관리합니다.
- 한 팔레트는 2~5개의 헥사코드(예: `#0F172A`, `#2563EB`) 문자열 조합과 팔레트 이름, 그리고 간단한 용도 설명을 저장합니다.
- UI 상에서 개별 색상 칩을 클릭하면 해당 헥사코드가 클립보드에 즉시 복사되어 파워포인트 색상 설정에 바로 붙여넣을 수 있어야 합니다.

---

## 3. 참고 사이트 관리 기능 (Reference Links)

### 요구사항 및 정책
- 기획 영감, 무료 아이콘 다운로드, PPT 무료 템플릿 제공 사이트 등을 관리합니다.
- 저장 모델은 사이트 명칭, **참고 사이트 URL**, 상세 설명으로 구성됩니다.
- **URL 노출 필수화**: 사용자가 사이트 카드를 보았을 때 사이트 URL이 명확히 표시되고, 클릭 시 해당 사이트로 즉시 새 창(target="_blank") 이동할 수 있어야 합니다.
